Detection (shared settings for all cameras)

Classes are in English, from the COCO set: person, car, truck, bus, bicycle, motorcycle, cat, dog, bird…
Motion gating: a cheap frame-difference check runs first, and YOLO fires only when something moves — so static objects aren't boxed and idle cameras cost almost no CPU. Lower threshold = more sensitive.
Record only moving objects: a parked car is still detected/boxed on the live view but won't start a recording — only an object with movement inside its box does (car driving past, person walking). Movement is measured relative to the whole frame, so rain, snow and night-time sensor flicker (which change the whole picture at once) don't trigger recording. Raise the box % if slight camera noise still triggers it.
Sustained motion before recording: the moving object must be seen for this many detection passes in a row (each ≈ the detect interval) before a recording starts — so a momentary rain streak, bug or IR flicker that clips an object for a single frame is ignored, while a real object keeps moving. The pre-buffer still backfills the seconds before, so nothing is lost. Set to 1 to record on the very first frame of motion; raise it if rain still triggers.
